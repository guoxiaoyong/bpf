
# Pin Assignment

## Example Module

```
module top(
	input  [2:0] BTN,
	input  [9:0] SW,
	output [9:0] LED,
	output [7:0] HLED0,
	output [7:0] HLED1,
	output [7:0] HLED2,
	output [7:0] HLED3
);

	assign LED[0] = 1'b1;
endmodule
```
